# Atividade Aula 09 – BDR

Consultas com INNER JOIN, LEFT JOIN, RIGHT JOIN e GROUP BY entre as tabelas do sistema de eventos.

## Conteúdo
- INNER JOIN entre evento e tipoevento
- INNER JOIN entre evento e localizacao
- LEFT JOIN: localizacao → evento → tipoevento
- RIGHT JOIN: evento → tipoevento → localizacao
- LEFT JOIN com COUNT e GROUP BY por cidade

## Ferramenta utilizada
pgAdmin 4
