# Atividade Aula 03 – BDR

Implementação do diagrama de banco de dados de uma locadora de veículos.

## Tabelas
- **cliente**: id, nome, cpf, telefone
- **carro**: placa, modelo, montadora, ano_fabricacao
- **aluguel**: id, cliente_id (FK), carro_placa (FK), data_inicio, data_fim

## Ferramenta utilizada
pgAdmin 4
