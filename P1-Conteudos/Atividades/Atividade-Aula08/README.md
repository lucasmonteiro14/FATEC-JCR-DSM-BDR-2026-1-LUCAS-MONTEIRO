# Atividade Aula 08 – BDR

Sistema bancário com transações e transferências usando BEGIN/COMMIT.

## Conteúdo
- Criação das tabelas: clientes, contas, transacao
- INSERTs de clientes e contas
- INNER JOIN entre contas e clientes
- Transferência entre contas com transação (BEGIN/COMMIT)

## Ferramenta utilizada
pgAdmin 4
